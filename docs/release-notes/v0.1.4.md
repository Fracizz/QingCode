### English

Multi-platform builds (unsigned macOS: right-click → Open the first time). WebView2 required on Windows.

#### Downloads

| Platform | File |
|----------|------|
| Windows x64 portable | `QingCode_0.1.4-windows-x64.exe` (also `QingCode_0.1.4.exe`) |
| Windows x64 installer | `QingCode_0.1.4-windows-x64-setup.exe` (also `QingCode_0.1.4-setup.exe`) |
| Windows ARM64 portable | `QingCode_0.1.4-windows-arm64.exe` |
| Windows ARM64 installer | `QingCode_0.1.4-windows-arm64-setup.exe` |
| macOS Apple Silicon | `QingCode_0.1.4-macos-arm64.dmg` / `.zip` |

#### Highlights

- **Editor minimap**: CodeGlance-style overview with syntax colors, Quick View, and `Ctrl+Shift+G`
- **Terminal**: floating find (`Ctrl+F`) with match count / highlights; clear buffer; default shell; clearer busy detection; profile / OpenCode lifecycle fixes; launch command tip on tab hover
- **Projects**: title-bar `+` opens a centered **Select Project** dialog (filter, restore hidden, open folder / temp project); activity-bar action zone can collapse
- **Settings & layout**: themed settings dropdowns; title-bar toggle between classic (bottom terminal) and side-terminal layouts
- **Accessibility & UI**: keyboard focus for project chips / editor & terminal tabs; Escape closes project dialog; Tooltip-only hover tips
- **Reliability**: terminal kill/write failures surface as toasts; Windows `pnpm tauri:dev` no longer dies when Vite is torn down by Tauri’s `beforeDevCommand`
- **Large files & trust**: size-tiered editing; workspace trust for untrusted projects
- **Formatting & encoding**: `Shift+Alt+F`, formatOnSave/Paste; multi-encoding open/save (incl. GBK/GB18030)
- **Multi-arch release**: Windows x64 / ARM64 (portable + NSIS), macOS Apple Silicon

### 中文

多平台构建（macOS 未签名：首次请右键 → 打开）。Windows 需安装 WebView2。

#### 下载

| 平台 | 文件 |
|------|------|
| Windows x64 便携 | `QingCode_0.1.4-windows-x64.exe`（兼 `QingCode_0.1.4.exe`） |
| Windows x64 安装包 | `QingCode_0.1.4-windows-x64-setup.exe`（兼 `QingCode_0.1.4-setup.exe`） |
| Windows ARM64 便携 | `QingCode_0.1.4-windows-arm64.exe` |
| Windows ARM64 安装包 | `QingCode_0.1.4-windows-arm64-setup.exe` |
| macOS Apple Silicon | `QingCode_0.1.4-macos-arm64.dmg` / `.zip` |

#### 主要变更

- **编辑器小地图**：CodeGlance 风格概览、语法着色、Quick View、`Ctrl+Shift+G` 开关
- **终端**：浮层查找（`Ctrl+F`，显示匹配数/高亮）、清空缓冲区、默认 Shell、忙碌检测、配置/OpenCode 生命周期；启动命令改到标签悬停提示
- **项目**：顶栏 `+` 打开居中「选择项目」对话框（筛选、恢复已隐藏、打开文件夹/临时项目）；活动栏动作区可折叠
- **设置与布局**：设置项主题化下拉；标题栏可在经典布局（底栏终端）与侧栏旁终端间切换
- **无障碍与交互**：项目 Chip / 编辑与终端标签可键盘操作；选择项目对话框支持 Escape；悬停统一用应用内 Tooltip
- **稳定性**：终端关闭/写入失败会 Toast；Windows 下 `pnpm tauri:dev` 不再因 Tauri 托管 Vite 异常退出而整条挂掉
- **大文件与信任**：按体积分级编辑；未信任项目受限
- **格式化与编码**：`Shift+Alt+F`、保存/粘贴格式化；多编码打开/保存（含 GBK/GB18030）
- **多架构发版**：Windows x64 / ARM64（便携 + NSIS）、macOS Apple Silicon

完整说明见 [CHANGELOG.md](https://github.com/Fracizz/QingCode/blob/master/CHANGELOG.md)。
