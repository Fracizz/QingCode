<a id="english"></a>

### English

Multi-platform builds (unsigned macOS: right-click → Open the first time). WebView2 required on Windows.

#### Downloads

| Platform | File |
|----------|------|
| Windows x64 portable | `QingCode_0.1.4-windows-x64.exe` (also `QingCode_0.1.4.exe`) |
| Windows x64 installer | `QingCode_0.1.4-windows-x64-setup.exe` (also `QingCode_0.1.4-setup.exe`) |
| Windows ARM64 portable | `QingCode_0.1.4-windows-arm64.exe` |
| Windows ARM64 installer | `QingCode_0.1.4-windows-arm64-setup.exe` |
| macOS Apple Silicon | `QingCode_0.1.4-macos-arm64.dmg` / `.zip` |

#### Highlights

- **Large-file editing tiers**: full / degraded / plain-text / read-only by size
- **Workspace trust**: restrict high-risk capabilities for untrusted projects
- **Formatting**: `Shift+Alt+F` plus formatOnSave / formatOnPaste
- **Exclude & encoding**: explorer/search excludes, gitignore filters, multi-encoding open/save (incl. GBK/GB18030)
- **Terminal**: scrollback persistence, cursor blink, collapse/close
- **Update check**: query Gitee/GitHub Releases and open the download page
- **Multi-arch release**: Windows x64 / ARM64 (portable + NSIS), macOS Apple Silicon
- **Fixes**: Git paths with Chinese/spaces/renames, Diff-theme white screen, root refresh clearing expanded folders, and more

Full notes: [CHANGELOG.en.md](https://github.com/Fracizz/QingCode/blob/master/CHANGELOG.en.md).

Gitee mirror: https://gitee.com/FrancizTest_admin/qing-code/releases

### 中文

多平台构建（macOS 未签名：首次请右键 → 打开）。Windows 需安装 WebView2。

#### 下载

| 平台 | 文件 |
|------|------|
| Windows x64 便携 | `QingCode_0.1.4-windows-x64.exe`（兼 `QingCode_0.1.4.exe`） |
| Windows x64 安装包 | `QingCode_0.1.4-windows-x64-setup.exe`（兼 `QingCode_0.1.4-setup.exe`） |
| Windows ARM64 便携 | `QingCode_0.1.4-windows-arm64.exe` |
| Windows ARM64 安装包 | `QingCode_0.1.4-windows-arm64-setup.exe` |
| macOS Apple Silicon | `QingCode_0.1.4-macos-arm64.dmg` / `.zip` |

#### 主要变更

- **大文件分级编辑**：按体积在完整 / 降级 / 纯文本 / 只读间切换，减轻超大文件卡顿
- **工作区信任**：未信任项目限制高风险能力
- **文档格式化**：`Shift+Alt+F` 与 formatOnSave / formatOnPaste（Prettier、rustfmt、shfmt、ruff/black、gofmt）
- **排除与编码**：`files.exclude` / `search.exclude`、gitignore 过滤、多编码打开/保存（含 GBK/GB18030）
- **终端**：滚动缓冲持久化、光标闪烁设置、收起/关闭
- **更新检查**：正式版可查询 Gitee/GitHub Release 并提示打开下载页
- **多架构发版**：Windows x64 / ARM64（便携 + NSIS）、macOS Apple Silicon
- **修复**：Git 中文/空格/重命名路径 Diff、Diff 主题启动白屏、根目录刷新清空已展开目录等

完整说明见 [CHANGELOG.md](https://github.com/Fracizz/QingCode/blob/master/CHANGELOG.md)。
