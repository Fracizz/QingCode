## QingCode v0.1.5

**Language / 语言:** [English](#english) · [中文](#中文)

<a id="english"></a>

### English

Multi-platform builds (unsigned macOS: right-click → Open the first time). WebView2 required on Windows.

#### Downloads

| Platform | File |
|----------|------|
| Windows x64 portable | `QingCode_0.1.5-windows-x64.exe` |
| Windows x64 installer | `QingCode_0.1.5-windows-x64-setup.exe` |
| Windows ARM64 portable | `QingCode_0.1.5-windows-arm64.exe` |
| Windows ARM64 installer | `QingCode_0.1.5-windows-arm64-setup.exe` |
| macOS Apple Silicon | `QingCode_0.1.5-macos-arm64.dmg` / `.zip` |

#### Highlights

- **Diff**: fixed MergeView scrolling (scrollbar works again); collapsed unchanged lines localized
- **Source Control**: expand untracked dirs to files; hover ~1s on truncated paths for the full path
- **Safety**: F5 / Ctrl+R no longer reload the whole WebView
- **Status bar**: process memory usage; tooltip arrow/spacing polish
- **Reliability**: session restore, explorer single-click open, Git panel performance with many changes

Full notes: [CHANGELOG.en.md](https://github.com/Fracizz/QingCode/blob/master/CHANGELOG.en.md).

Gitee mirror: https://gitee.com/FrancizTest_admin/qing-code/releases

<a id="中文"></a>

### 中文

多平台构建（macOS 未签名：首次请右键 → 打开）。Windows 需安装 WebView2。

#### 下载

| 平台 | 文件 |
|------|------|
| Windows x64 便携 | `QingCode_0.1.5-windows-x64.exe` |
| Windows x64 安装包 | `QingCode_0.1.5-windows-x64-setup.exe` |
| Windows ARM64 便携 | `QingCode_0.1.5-windows-arm64.exe` |
| Windows ARM64 安装包 | `QingCode_0.1.5-windows-arm64-setup.exe` |
| macOS Apple Silicon | `QingCode_0.1.5-macos-arm64.dmg` / `.zip` |

#### 主要变更

- **差异对比**：修复侧栏无法滚动；折叠未改动行文案国际化（「N 行未更改」）
- **源代码管理**：展开未跟踪目录内的文件；截断路径悬停约 1 秒显示完整路径
- **安全体验**：禁用 F5 / Ctrl+R 触发 WebView 整页刷新
- **状态栏**：显示进程内存占用；Tooltip 箭头与间距优化
- **稳定性**：会话恢复、资源管理器单击打开、大量变更时 Git 面板性能

完整说明见 [CHANGELOG.md](https://github.com/Fracizz/QingCode/blob/master/CHANGELOG.md)。
