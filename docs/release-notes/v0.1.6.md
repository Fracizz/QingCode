### English

Multi-platform builds (unsigned macOS: right-click → Open the first time). WebView2 required on Windows.

#### Downloads

| Platform | File |
|----------|------|
| Windows x64 portable | `QingCode_0.1.6-windows-x64.exe` |
| Windows x64 installer | `QingCode_0.1.6-windows-x64-setup.exe` |
| Windows ARM64 portable | `QingCode_0.1.6-windows-arm64.exe` |
| Windows ARM64 installer | `QingCode_0.1.6-windows-arm64-setup.exe` |
| macOS Apple Silicon | `QingCode_0.1.6-macos-arm64.dmg` / `.zip` |

#### Highlights

- **Layout**: side terminal splits evenly with editor; classic ↔ side switch keeps terminal mounted
- **Editor**: YAML, TOML, Shell, Rust, Go syntax highlighting; bracket/indent guide fixes
- **Search**: mixed layout and path quick-open polish
- **Source Control**: full-page SCM, Git pull/conflicts, batch actions, history layout
- **Terminal**: Windows shell auto-select; drag/layout-switch stability
- **Release**: installer embeds latest frontend; update check can download installer directly

### 中文

多平台构建（macOS 未签名：首次请右键 → 打开）。Windows 需安装 WebView2。

#### 下载

| 平台 | 文件 |
|------|------|
| Windows x64 便携 | `QingCode_0.1.6-windows-x64.exe` |
| Windows x64 安装包 | `QingCode_0.1.6-windows-x64-setup.exe` |
| Windows ARM64 便携 | `QingCode_0.1.6-windows-arm64.exe` |
| Windows ARM64 安装包 | `QingCode_0.1.6-windows-arm64-setup.exe` |

#### 主要变更

- **布局**：侧栏旁终端与编辑器默认均分；经典/侧栏切换保持终端挂载、减少卡顿
- **编辑器**：YAML/TOML/Shell/Rust/Go 语法高亮；括号/缩进参考线与行号区样式修复
- **搜索**：混排布局与路径快速打开完善
- **源代码管理**：整页 SCM、Git 拉取/冲突、批量操作与历史布局
- **终端**：Windows Shell 自动选择；拖动与布局切换稳定性
- **发版**：安装包嵌入最新前端；检查更新可直接下载安装包
